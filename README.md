# Deep Learning (AIL721 / AIL7021)

This repository contains assignments, implementations, reports, experimental results, and ablation studies completed as part of the Deep Learning course at IIT Delhi during Semester II, 2025–26.

# Assignment 1

## Part A: Neural Network Regression

- Implemented a fully connected neural network from scratch using NumPy.
- Implemented forward propagation, backward propagation, and Mini-Batch SGD.
- Performed experiments with different hidden layer sizes, network depths, and activation functions.
- Compared the from-scratch implementation with Scikit-Learn's MLPRegressor.
- Evaluated training loss, validation loss, convergence behavior, and training time.

## Part B: Neural Network Classification

- Implemented neural network classifiers for Forest Cover Type prediction.
- Developed both NumPy-based and PyTorch-based implementations.
- Performed data preprocessing, model training, evaluation, and comparison.
- Evaluated performance using Macro-F1 Score.

# Assignment 2

## Task 1: Bird Species Classification

- Developed CNN-based image classification models using PyTorch.
- Classified images belonging to 15 bird species commonly found in India.
- Applied data augmentation, regularization techniques, learning rate scheduling, and hyperparameter tuning.
- Evaluated performance using Macro-F1 and Micro-F1 scores.

## Task 2: Multi-Class Cell Segmentation

- Developed semantic segmentation models for histopathology images.
- Performed pixel-wise classification of cell nuclei.
- Implemented CNN/UNet-based segmentation architectures.
- Evaluated segmentation performance using Mean Intersection over Union (mIoU) and Mean Dice Score.

---

# Assignment 3

## Task 1: Sentiment Analysis

- Implemented a Vanilla RNN from scratch using NumPy.
- Implemented Backpropagation Through Time (BPTT).
- Developed a PyTorch-based RNN model for comparison.
- Used GloVe embeddings for text representation.
- Performed sentiment classification and comparative analysis.

## Task 2: Transformer Implementation

- Implemented a Decoder-Only Transformer from scratch.
- Implemented Multi-Head Self-Attention, Positional Encoding, Masked Self-Attention, and Beam Search Decoding.
- Conducted multiple ablation studies including attention heads, trainable embeddings, context window size, layer normalization, and beam search size.
- Evaluated model performance using Perplexity and Loss metrics.

## Task 3: Graph Convolutional Network (GCN)

- Implemented a Graph Convolutional Network from scratch using NumPy.
- Performed multi-label node classification on the Protein-Protein Interaction (PPI) dataset.
- Implemented graph convolution layers, message passing, neighborhood aggregation, Binary Cross Entropy loss, and Adam optimization.
- Conducted ablation studies and performance analysis.

---

# Assignment 4

## Task 1: Reinforcement Learning

- Improved an Actor-Critic agent for Atari Pong.
- Implemented CNN-based feature extraction, frame stacking, entropy regularization, advantage normalization, gradient clipping, and parallel environment training.
- Compared performance against the provided baseline agent.
- Evaluated using Mean Episode Return, Win Rate, and Learning Speed.

## Task 2: Variational Autoencoders (VAE)

- Implemented MLP-VAE and CNN β-VAE architectures.
- Performed representation learning on the Hindi MNIST dataset.
- Conducted latent space analysis, interpolation, reconstruction quality evaluation, and sample generation.
- Evaluated performance using FID (Fréchet Inception Distance) and SSIM (Structural Similarity Index).
- Performed ablation studies on latent dimensions, CNN architectures, and β-VAE configurations.
